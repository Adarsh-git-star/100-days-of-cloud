# Day 3 - Linux Navigation & Output Commands

##  Commands Learned

###  cd Command Options
- cd ..  → Move up one directory
- cd ~   → Go to home directory
- cd -   → Previous directory
- cd /   → Root directory

###  pwd Command Options
- pwd -L → Logical path
- pwd -P → Physical path

###  echo Command Options
- echo -n "Hello" → No new line
- echo -e "Hello\nWorld" → New line
- echo -E "Hello\nWorld" → Disable special characters

---

##  What I Learned
- Navigation using cd is very important  
- pwd helps to know exact location  
- echo is useful for output and scripting
